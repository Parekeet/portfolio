Mint Thai

Mint is a website for Mint Thai, a fictional Thai Restaurant in Houston, Texas. Users will be able to browse through the site and place an order for pick up.

Navigate through different tabs to read about the restaurant. Once users are ready to place an order, they can navigate to the 'Order' tab. Users must be logged in to place an order. Once logged in, users can create, save, view, edit, and delete customized orders. Once they've completed making their order, submit it and pick it up!

Technologies Used:

HTML | CSS | Ruby | Ruby on Rails | Bootstrap

VURS

VURS

Our group project. VURS is a user generated song recommendation application. Users are able to search for music, as well has play, pause, mute, unmute, and skip songs using our own audio player.

Users can log in with their SoundCloud credentials and import their playlist from SoundCloud to VURS. Doing so allows users to upvote songs and make recommendations for other users to see. The top three recommended songs for each song will be displayed.

Technologies Used:

HTML | CSS | JavaScript | jQuery | Materialize | Mongoose | Express | Node.js | MongoDB | lo-dash | Passport/OAuth

3P API: SoundCloud API

Mint Thai

A MEAN Stack version of Mint Thai.

When not logged in, users are able to browse the site's menu, about us, and contact page. From the contact page, users can find directions to the restaurant by clicking on the map. Users also have the option of getting in contact with the owner by clicking on the email icon. Once logged in, users have the ability to place orders for a pick up order. Once the users submit their orders, both the restaurant owner and the users will receive an email notification about said order.
